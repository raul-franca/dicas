
config basicas: 
$ git config --global user.name "Fulano da Silva"
$ git config --global user.email fulanodasilva.git@gmail.com

Criando um repositório:
dentro da pasta do novo repositório
$ git init

Rastreando o arquivo:

$ git status

Para que o arquivo seja rastreado, devemos executar o seguinte comando:

$ git add {nome do arquivo} 

Para rastrearmos a modificação, devemos executar o comando git add novamente:

$ git add {nome do arquivo}

Rastreando e gravando as alterações no repositório:

para rastrear a modificação 
$ git add {nome do arquivo}

Com a modificação rastreada, podemos gravá-la no repositório,
com o comando git commit:

$ git commit -m "descrição da modificação"

Verificando alterações realizadas:

$ git log

Apontando seu projeto para o GitHub

$ git remote add origin https://github.com/raul-franca/dicas.git

Enviando as alterações para o GitHub:

$ git push origin master

Obtendo projeto do GitHub:

$ git clone {endereço do gitHub}
