git push <destino> master 


config básicas: 
 git config --global user.name "Raul França"
 git config --global user.email raulmfranca@gmail.com

Para visualizar as configurações : 
 git config --list

Criando um novo repositório:
 git init

Mostra o status dos arquivos: 
 git status 

Adiciona a o monitoramento: 
 git add <file> ou 
 git add -A <-- add todos os arquivos de uma vez

Para dar um commit: 
 git commit -m “[descriptive message]”
 git commit -am “[descriptive message]” <-- add e commit !cuidado na hora do git push ele não manda novos arquivos 

Ver commits:
git log
